# Soutěžní Had
Toto je můj pokus o zlepšení klasického hada do školní soutěže.

Vy hrajete za zeleného hada. Zbírejte jídlo a vyhýbejte se zlým oranžovým hadům. Pokud se vás dotknou tak zemřete v těchto případech zmáčkněte R a zkuste to znovu. 
Aby vám to šlo lépe, použíte mezerník pro přesun mezi stíny, tam vás hadi nekousnou, ale tato cesta vás může stát i život.
